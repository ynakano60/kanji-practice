# 漢字れんしゅう

小学校の漢字テスト練習プリントを取り込んで、スマホで手書き練習できる Web アプリ(PWA)です。

## 特長

- **スマホ最適化**: 画面上部に問題、下部に手書きマス。指で漢字を書いて練習できます
- **3つの取り込み方法**
  - 📷 **写真から(AI)**: プリントを撮影すると Claude API が問題と正解を自動抽出(要 API キー)
  - 📋 **テキスト貼り付け**: スマホ標準の OCR(iPhone のテキスト認識表示 / Google レンズ)でコピーした文字を貼り付けて自動整形(無料・キー不要)
  - ✏️ **手入力**: 1問ずつフォームで入力
- **自己採点と復習**: 答え合わせで ○× を記録し、まちがえた問題だけを繰り返し練習できます
- **データは端末内に保存**: 問題と成績は削除するまでブラウザ(IndexedDB)に保存。共有サーバーはなく、各家庭のデータは完全に独立です
- **オフライン対応**: 一度開けば、練習はネット接続なしでも使えます
- **バックアップ**: JSON ファイルの書き出し/読み込みで機種変更にも対応

## 使い方

1. スマホのブラウザでアプリの URL を開く
2. 「ホーム画面に追加」でアプリとしてインストール(推奨)
3. 「＋ 新しい問題を取り込む」からプリントを登録
4. 取り込み内容を確認・修正して保存
5. 練習開始!(番号順 / ランダム / まちがいのみ)

## 写真取り込み(AI)を使う場合

1. [Anthropic Console](https://console.anthropic.com/) でアカウントを作成し、API キーを取得(クレジットの事前購入が必要。写真1枚の読み取りは数円程度)
2. アプリの「⚙ 設定」で API キーを貼り付けて保存

API キーはお使いの端末の localStorage にのみ保存され、外部に送信されるのは Anthropic API の呼び出し時だけです。

## 開発

ビルド不要の静的サイトです(Vanilla JS + PWA)。ローカルで動かすには:

```sh
npx serve .
# または
python -m http.server 8000
```

### 構成

| ファイル | 役割 |
|---|---|
| `index.html` / `style.css` | 画面とスタイル |
| `js/app.js` | 画面遷移・練習画面(手書きキャンバス)・設定 |
| `js/db.js` | IndexedDB(問題セット・成績の永続化) |
| `js/parser.js` | OCR 貼り付けテキストの整形 |
| `js/api.js` | Claude API 呼び出し(写真→問題の抽出) |
| `sw.js` | Service Worker(オフライン対応) |

デプロイは GitHub Pages(main ブランチのルート)を想定しています。`sw.js` の `CACHE_VERSION` を上げるとキャッシュが更新されます。

要件定義は [要件定義.md](./要件定義.md) を参照してください。
